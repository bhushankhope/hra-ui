src/lib/cell-summary/cell-summary.model.ts

CELL_SCHEMA
Default value : z.object({ cell: z.object({ id: z.string(), label: z.string(), }), biomarker: z.object({ id: z.string(), label: z.string(), }), count: z.number(), percentage: z.number(), })

The CELL_SCHEMA has an object that contains an id and a label for the cell, another object that contains an id and a label for the biomarker, and a count and a percentage that are both numbers.
CELL_SUMMARY_AGGREGATE_SCHEMA
Default value : z.record( z.object({ label: z.string(), columns: z.string().array(), rows: z .tuple([z.string(), z.number().optional()]) .rest( z .object({ color: z.string(), size: z.number(), data: CELL_SCHEMA, }) .optional() ) .array(), }) )

The CELL_SUMMARY_AGGREGATE_SCHEMA is a record of Aggregate objects, where each Aggregate object contains a label, an array of columns that are strings, and an array of rows that are AggregateRow objects.
CELL_SUMMARY_SCHEMA
Default value : z.record( z.object({ label: z.string(), entries: CELL_SCHEMA.array(), }) )

The CELL_SUMMARY_SCHEMA constant is a record of CellSummary objects, where each CellSummary object contains a label and an array of CELL_SCHEMA objects.

src/lib/download/download.model.ts

DOWNLOAD_ENTRY
Default value : z.discriminatedUnion('type', [ z.object({ type: z.literal('url'), url: z.string() }), z.object({ type: z.literal('data'), data: z.string() }), ])

Define a Zod schema for DOWNLOAD_ENTRY, which is a discriminated union of two objects with different properties
DOWNLOAD_FORMAT
Default value : z .object({ id: DOWNLOAD_FORMAT_ID, label: z.string(), extension: z.string(), }) .partial({ extension: true })

Define a Zod schema for DOWNLOAD_FORMAT, which is an object with id, label, and extension properties
DOWNLOAD_FORMAT_ID
Default value : z .string() .transform((id) => `DownloadFormatId:'${id}'`) .brand('DownloadFormatId')

Define a Zod schema for DOWNLOAD_FORMAT_ID, which is a string that is transformed into a branded string
DOWNLOAD_MODEL
Default value : z.object({ formats: z.record(DOWNLOAD_FORMAT_ID, DOWNLOAD_FORMAT), entries: z.record(DOWNLOAD_FORMAT_ID, DOWNLOAD_ENTRY), })

Define a Zod schema for DOWNLOAD_MODEL, which is an object with formats and entries properties
