src/lib/download/download.model.ts

DOWNLOAD_ENTRY
Default value : z.discriminatedUnion('type', [ z.object({ type: z.literal('url'), url: z.string() }), z.object({ type: z.literal('data'), data: z.string() }), ])

Define a Zod schema for DOWNLOAD_ENTRY, which is a discriminated union of two objects with different properties
DOWNLOAD_FORMAT
Default value : z .object({ id: DOWNLOAD_FORMAT_ID, label: z.string(), extension: z.string(), }) .partial({ extension: true })

Define a Zod schema for DOWNLOAD_FORMAT, which is an object with id, label, and extension properties
DOWNLOAD_FORMAT_ID
Default value : z .string() .transform((id) => `DownloadFormatId:'${id}'`) .brand('DownloadFormatId')

Define a Zod schema for DOWNLOAD_FORMAT_ID, which is a string that is transformed into a branded string
DOWNLOAD_MODEL
Default value : z.object({ formats: z.record(DOWNLOAD_FORMAT_ID, DOWNLOAD_FORMAT), entries: z.record(DOWNLOAD_FORMAT_ID, DOWNLOAD_ENTRY), })

Define a Zod schema for DOWNLOAD_MODEL, which is an object with formats and entries properties
