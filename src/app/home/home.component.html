ASCT+B REPORTER

The CCF ASCT+B Reporter is a basic visualization tool for displaying the flattened ASCT+B tables built using Angular.



Features
Visualizations

The ASCT+B Reporter houses two different types of visualizations. One being the partonomy tree (Anatomical Structures) combined with a bimidal network (Cell Types and Biomarkers). The other one being an Indented List for a more traditional, hierachical look.

Report Generator

The ASCT+B Reporter containes a download-able report generator that lists various statistics of the visualization, conputed from the flattened organ sheet.

Debug Log

The Debug Log enlists any warnings or errors that could've occurred during the creation of the visualization.



Functions

The nodes in the bimodal network can be sorted by their relative sizes and alphabetically. The size represents the degree of node. The nodes can further sized by their indegrees and outdegrees.

Exports

The tree visualization can be exported to an image in either PNG or vector (SVG) format.
